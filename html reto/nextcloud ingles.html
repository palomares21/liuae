NEXTCLOUD

Before starting with the installation of nextcloud we have to do an apt update y upgrade. After the update and upgrade we do an apt install apache2 php php-mysql mysql-server.

When we have everything installed we put in sudo and put mysql -u root. Now we will proceed to create a database create database nextcloud; do not forget to put ; at the end of each command.Having the database created now we will have to create a user with password create user (nombre) identified by (contraseña); now we will proceed to give permissions grant all privileges on *.* to (nombre) with grant option;
flush privileges;
to exit we put exit.

having done all the above we go to the folder html to get to it we put cd /var/www/html now we install nextcloud wget https://download.nextcloud.com/server/releases/nextcloud-23.0.0.zip.At this moment to decompress the zip file we will have to install the unzip command for it we put apt install unzip.Having installed the unzip we proceed to unzip the fileunzip nextcloud-23.0.0.zip having put this command now we will have a folder in /var/www/html called nextcloud.


It is time to give permissions and for this we put chown www-data:www-data nextcloud -R then you put chmod 775 nextcloud -R.

Now we go to duckdns and create a domain.

Having created the domain we go to cd /etc/apache2/sites-enabled here we will have a file called 000-default.conf. Where it says domain name we put the address of our duckdns and next to html we put nextcloud.

Now we are going to increase the storage of nextcloud go to nano /etc/php/7.4/apache/php.ini and in this file look for post_max_size and increase it and also look for file_upload_max_size and increase it and then save the file do a systemctl restart apache2 and now go to google and put our duckdns address.Now we install everything it asks for.

After installing it we fill in the data and we are done.
