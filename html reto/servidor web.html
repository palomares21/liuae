SERVIDOR WEB

Pones en la terminal cd /var/www/html para acceder al index.html pero primero crea el archivo con nano index.html

Para entrar al index.html escribes nano index.html ejemplo=

Entras en /etc/resolv.conf pones nameserver 8.8.8.8 y nameserver 8.8.4.4

Luego instalamos certbot: apt install certbot apt install python3-certbot-apache después de instalar certbot entramos en /etc/apache2/sites-enabled/ y nos metemos en el archivo 000-default conf descomentamos el server name y ponemos la dirección del duckdns te tiene que quedar más o menos como en la foto.

Luego ponemos sudo certbot –apache y sigues los pasos: 1. a@a.com 2. A 3. n 4. le das al enter 5. para terminar pones 2 y así conseguiremos el certificado.
