WEB SITE

You put in the terminal cd /var/www/html to access the index.html but first create the file with nano index.html

To enter the index.html you type nano index.html example=

Go into /etc/resolv.conf put nameserver 8.8.8.8 and nameserver 8.8.4.4

Then install certbot: apt install certbot apt install python3-certbot-apache After installing certbot go into /etc/apache2/sites-enabled/ and get into the 000-default.conf file unzip the server name and put the duckdns address, it should look more or less like the picture.

Then put sudo certbot –apache and follow the steps: 1. a@a.com 2. A 3. n 4. hit enter 5. to finish you put 2 and this way we will get the certificate.
