Seguridad

Lo primero que hay que hacer es un apt update & apt upgrade

Cuando tengamos todo instalado nos ponemos en sudo y ponemos mysql -u root. Ahora procederemos a crear una base de datos create database nextcloud; no olvides poner ; al final de cada comando.Teniendo la base de datos creada ahora tendremos que crear un usuario con contraseña create user (nombre) identified by (contraseña); ahora procederemos a darle permisos grant all privileges on *.* to (nombre) with grant option;
flush privileges;
para salir ponemos exit.

Habiendo hecho todo lo anterior vamos a la carpeta html para llegar a ella ponemos cd /var/www/html ahora instalamos nextcloud wget https://download.nextcloud.com/server/releases/nextcloud-23.0.0.zip.En este momento para descomprimir el archivo zip deberemos de instalar el comando unzip para ello ponemos apt install unzip.Teniendo instalado el unzip procedemos a descomprimir el archivo unzip nextcloud-23.0.0.zip habiendo puesto este comando ahora tendremos una carpeta en /var/www/html llamada nextcloud.


Toca dar permisos y para ello ponemos chown www-data:www-data nextcloud -R después pones chmod 775 nextcloud -R.

Ahora vamos al duckdns y creamos un dominio

Habiendo creado el dominio vamos a cd /etc/apache2/sites-enabled aquí tendremos un archivo llamado 000-default.conf. Donde pone domain name ponemos la dirección de nuestro duckdns y al lado de html ponemos nextcloud.

ahora vamos a aumentar el almacenamiento de nextcloud entra a nano /etc/php/7.4/apache/php.ini y en este archivo buscamos post_max_size y lo aumentamos y también buscamos file_upload_max_size y lo aumentas y luego guardamos el archivo hacemos un systemctl restart apache2 y ahora vamos a google y ponemos nuestra dirección de duckdns.Ahora instalamos todo lo que nos pide.
